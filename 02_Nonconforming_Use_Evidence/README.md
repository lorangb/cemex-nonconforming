# 02 — Nonconforming Use Evidence  (core of the appeal)
Whether structures and uses added to the CEMEX site *after* it became a legal
nonconforming use were permissible.

- **New_Structures/** — permits/records for towers (Qwest, Verizon, T-Mobile), dust
  collectors, silos, signs, electrical upgrades, wells; the 2007 Notice of Violation;
  the Sept 2023 visual evidence; and the "structural enlargement" correspondence.
- **County_Determinations/** — the 1967 variance, the 2002 nonconforming-use confirmation
  letters, the SU-96-18 removal-of-mining record, land-use letters, resolutions.
- **Board_of_Adjustment/** — the Board's rules, the "Ask a Planner" exchange, the CEMEX
  hearing materials, and **Martin_Marietta_Precedent/** (a different company's BOA case
  kept separate so it is not mistaken for CEMEX's own record).
- **Precedent_Sierra_v_Billingsley/** — the controlling local precedent.
